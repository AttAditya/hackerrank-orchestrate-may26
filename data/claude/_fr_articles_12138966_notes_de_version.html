Compétences pour les organisations, les partenaires et l'écosystème

En octobre, nous avons introduit les compétences—un moyen d'enseigner à Claude des flux de travail répétables adaptés à votre façon de travailler. Les compétences sont maintenant plus faciles à déployer, découvrir et créer avec la gestion à l'échelle de l'organisation pour les plans Team et Enterprise, un répertoire de compétences créées par des partenaires, et une norme ouverte (Agent Skills) afin que les compétences fonctionnent sur les plateformes d'IA.

Mises à jour de Claude dans Chrome

Claude dans Chrome est maintenant disponible en version bêta pour tous les abonnés aux plans payants, y compris les plans Pro, Team et Enterprise.

  • Intégration de Claude Code : Créez dans votre terminal avec Claude Code, puis testez et vérifiez dans le navigateur avec l'extension Chrome. Claude peut lire les erreurs de console, les requêtes réseau et l'état du DOM pour aider à déboguer les problèmes directement.

  • Contrôler les actions du navigateur depuis Claude Desktop : Commencez une tâche dans Claude Desktop et laissez-la gérer le travail dans le navigateur sans changer de fenêtre.

  • Enregistrer un flux de travail : Enseignez à Claude votre flux de travail en enregistrant les étapes, et Claude apprend à les répéter. Utile pour les tâches de navigateur répétitives que vous souhaitez déléguer.

  • Journaux de console : Claude peut maintenant lire la sortie de la console du navigateur, aidant les développeurs à identifier et corriger les erreurs sans changer de contexte.

  • Contrôles d'administration pour Team et Enterprise : Les administrateurs peuvent maintenant gérer Claude dans Chrome au niveau de l'organisation :

    • Activer ou désactiver l'extension à l'échelle de l'organisation

    • Configurer les listes blanches et les listes noires pour contrôler les sites auxquels Claude peut accéder

    • Gérer Claude dans Chrome dans Paramètres d'administration > Extension de navigateur

Mises à jour de Claude dans Chrome

Claude dans Chrome est maintenant disponible en bêta pour tous les abonnés au plan Max. Après des mois de tests, nous élargissons l'accès avec les mises à jour suivantes :

Mises à jour de Claude dans Chrome

Les utilisateurs Max restants sur notre liste d'attente ont reçu l'accès à Claude dans Chrome, ainsi que les mises à jour suivantes :

  • Alimenté par Sonnet 4.5 : Claude dans Chrome utilise maintenant par défaut Sonnet 4.5, notre modèle le plus intelligent à ce jour. Amélioré pour les tâches de navigateur, vous remarquerez un meilleur raisonnement, moins d'erreurs et une exécution des tâches plus fiable—en particulier pour les flux de travail multi-étapes.

  • Travailler sur plusieurs onglets : Claude peut maintenant gérer plusieurs onglets de navigateur à la fois. Faites simplement glisser les onglets dans le groupe d'onglets de Claude et il peut voir et travailler sur tous simultanément—plus besoin de sauter d'avant en arrière pour rassembler les informations avant d'agir.

  • Plus intelligent sur les sites que vous utilisez chaque jour : Claude sait maintenant comment naviguer sur Slack, Google Calendar, Gmail, Google Docs et GitHub sans que vous ayez à diriger chaque clic. Demandez à Claude de « planifier une réunion » ou « mettre à jour le document » et il sait quoi faire—pas besoin de lui expliquer chaque étape. Nous continuons à améliorer la compréhension de Claude pour d'autres sites populaires.

  • Recevez une notification quand Claude a besoin de vous : Activez les notifications et Claude vous préviendra quand il a besoin de votre permission ou quand une tâche est terminée. Maintenant, vous pouvez passer à d'autres travaux pendant que Claude gère les choses en arrière-plan.

  • Montrez à Claude exactement ce que vous voulez dire : Téléchargez des images ou prenez des captures d'écran de zones d'écran spécifiques pour donner à Claude un contexte visuel précis. Beaucoup plus rapide que de décrire ce que vous regardez.
