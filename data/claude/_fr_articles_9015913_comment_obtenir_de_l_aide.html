Comment obtenir de l'aide

12 mars 2026
Table des matières

Ce guide explique comment obtenir de l'aide pour votre compte Claude ou Console. Différentes options d'assistance sont disponibles selon votre plan.

Plans Pro et Max, propriétaires de plans Team et Enterprise, et administrateurs Console

Vous avez un accès complet à :

  • Toute la documentation d'aide

  • Fin, notre agent d'assistance IA

  • Une assistance supplémentaire de notre équipe d'assistance produit

Remarque : Bien que nous n'offrons pas de support téléphonique ou de chat en direct, notre équipe d'assistance produit sera heureuse de vous aider via notre messagerie d'assistance.

Demander de l'aide pour Claude

  1. Connectez-vous à votre compte Claude.

  2. Cliquez sur vos initiales ou votre nom dans le coin inférieur gauche et sélectionnez « Obtenir de l'aide » dans le menu.

  3. Dans la messagerie d'assistance :

    • Recherchez les ressources d'aide

    • Discutez avec Fin, notre agent d'assistance IA

  4. Cliquez sur « Nous envoyer un message » pour commencer à discuter avec Fin.

  5. Cliquez sur « Accepter », puis sélectionnez votre problème parmi les options disponibles.

  6. Selon le problème sélectionné, fournissez des détails supplémentaires lorsque vous y êtes invité.

  7. Si votre problème nécessite une enquête supplémentaire ou une contribution de notre équipe d'assistance produit, Fin transmettra votre demande et quelqu'un vous répondra par e-mail.

Demander de l'aide pour l'API Claude ou Console

  1. Connectez-vous à votre compte Console (platform.claude.com).

  2. Cliquez sur vos initiales ou votre nom dans le coin inférieur gauche de Console et sélectionnez « Obtenir de l'aide » dans le menu.

  3. Cela ouvrira une messagerie d'assistance dans le coin inférieur droit de Console.

  4. Dans la messagerie d'assistance :

    • Recherchez les ressources d'aide

    • Discutez avec Fin, notre agent d'assistance IA

  5. Cliquez sur « Nous envoyer un message » pour commencer à discuter avec Fin.

  6. Cliquez sur « Accepter », puis sélectionnez votre problème parmi les options disponibles.

  7. Selon le problème sélectionné, fournissez des détails supplémentaires lorsque vous y êtes invité.

  8. Si votre problème nécessite une enquête supplémentaire ou une contribution de notre équipe d'assistance produit, Fin transmettra votre demande et quelqu'un vous répondra par e-mail.

Demander de l'aide pour tout type de compte depuis le Centre d'aide

Les utilisateurs connectés à leurs comptes Claude ou Console pourront cliquer sur l'icône dans le coin inférieur droit de n'importe quelle page du Centre d'aide pour ouvrir la messagerie d'assistance et commencer à discuter avec Fin.

Important : Pour les utilisateurs de Console, cela s'applique spécifiquement à ceux connectés via platform.claude.com. Si vous êtes connecté via console.anthropic.com, vous serez invité à visiter platform.claude.com pour vous authentifier avant d'envoyer un message dans la messagerie d'assistance du Centre d'aide.

Non-propriétaires de plans Team et Enterprise, et non-administrateurs Console

Vous discuterez avec Fin, un agent d'assistance IA, pour vous aider à répondre à vos questions. Le support spécialisé humain n'est pas directement disponible pour votre type de siège de compte. Si Fin ne peut pas trouver de solution et que vous avez besoin d'escalader vers notre équipe humaine de spécialistes, le propriétaire principal, le propriétaire ou l'administrateur Console de votre compte peut vous contacter en votre nom.

Propriétaires de plans Enterprise

Si vous êtes le propriétaire principal ou propriétaire d'un plan Claude for Work Enterprise, vous pouvez également soumettre une demande via le formulaire d'assistance Enterprise.

Remarque : Vous devez être connecté à votre compte de propriétaire pour accéder à ce formulaire.

Utilisateurs gratuits de Claude

Les utilisateurs gratuits de Claude.ai ont accès à :

  • Toute la documentation d'aide

  • Fin, notre agent d'assistance IA

  • Assistance pour la suppression de compte dans les cas où l'assistance en libre-service n'est pas disponible

Pour obtenir de l'aide :

  1. Connectez-vous à votre compte Claude.ai.

  2. Cliquez sur vos initiales ou votre nom dans le coin inférieur gauche.

  3. Sélectionnez « Obtenir de l'aide ».

  4. Utilisez la messagerie d'assistance pour :

    • Rechercher les ressources d'aide.

    • Discuter avec Fin, notre agent d'assistance IA.

Impossible de se connecter ?

Si vous ne pouvez pas accéder à votre compte, nous pouvons toujours vous aider avec :

  1. Suppression de compte

  2. Exports de données

  3. Assistance aux abonnements

Pour demander une assistance :

  1. Cliquez sur l'icône de message dans le coin inférieur droit du Centre d'aide.

  2. Commencez une conversation.

  3. Sélectionnez « Je ne peux pas me connecter ».

À propos de l'assistance aux abonnements

Si vous êtes un abonné payant ayant des difficultés à accéder à votre compte, notre équipe peut :

  • Vous aider à localiser les informations de connexion

  • Traiter les annulations d'abonnement

  • Vérifier l'admissibilité au remboursement

Remarque : Pour vous aider avec ces tâches, nous devrons valider les détails de votre compte et le mode de paiement utilisé.

Point de contact désigné pour les utilisateurs de l'UE

Les utilisateurs de l'UE peuvent trouver les instructions sur la façon de trouver le point de contact unique pour les utilisateurs de l'UE en vertu du DSA ici.

  • Les utilisateurs en Australie peuvent trouver les instructions pour signaler le non-respect de la norme DIS et des informations sur le rôle et les fonctions du commissaire à la sécurité en ligne ici.
